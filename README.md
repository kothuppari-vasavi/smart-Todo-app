# Smart To-Do List Web App 🌐

## 📌 Description
A web-based To-Do application built using Flask with task management features.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Data stored using SQLite

## 🛠️ Tech Used
- Python
- Flask
- HTML
- CSS
- SQLite

## ▶️ How to Run
pip install flask  
python app.py
